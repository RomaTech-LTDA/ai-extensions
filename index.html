@romatech/ai-extensions

GitHub npm

Plug-and-play AI enablement framework for Node.js APIs. Transforms Express/Fastify/Koa endpoints into MCP-compatible tool providers and RAG-enabled knowledge sources — without architectural rewrites.

Installation

npm install @romatech/ai-extensions

Quick Start

import express from 'express';
import { useAi, registerAiMetadata, aiTool, aiHidden } from '@romatech/ai-extensions';

const app = express();
app.use(express.json());

app.post('/api/orders', (req, res) => {
  res.json({ id: 2, status: 'Created', ...req.body });
});

app.delete('/api/orders/:id', (req, res) => {
  res.sendStatus(204);
});

// Annotate with AI metadata
registerAiMetadata('POST', '/api/orders', aiTool({
  toolName: 'create_order',
  description: 'Creates a new customer order',
  category: 'Orders',
  rateLimit: 10,
}));

registerAiMetadata('DELETE', '/api/orders/:id', aiHidden());

// Enable AI
useAi(app, { baseUrl: 'http://localhost:3000' });
app.listen(3000);

Features

  • MCP Server — Expose your API endpoints as MCP tools for LLMs
  • RAG Search — Semantic search across API documentation
  • Metadata Decorators — aiTool(), aiHidden(), aiReadOnly()
  • Rate Limiting — Built-in sliding-window rate limiter
  • Multi-Framework — Express, Fastify, and Koa adapters
  • SSE Transport — Server-Sent Events for persistent connections
  • MCP Resources — Read-only endpoints exposed as MCP resources
  • Metrics — Built-in tool usage tracking (call count, latency, errors)
  • Authentication — API key auth via Bearer token with optional header forwarding
  • CORS — Configurable CORS headers for cross-origin MCP clients
  • Zero External Dependencies — No OpenAI/Ollama required

MCP Protocol

# Initialize
POST /mcp  {"jsonrpc":"2.0","id":1,"method":"initialize"}

# List tools
POST /mcp  {"jsonrpc":"2.0","id":2,"method":"tools/list"}

# Call a tool
POST /mcp  {"jsonrpc":"2.0","id":3,"method":"tools/call","params":{"name":"create_order","arguments":{...}}}
