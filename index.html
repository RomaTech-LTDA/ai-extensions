@romatech/ai-extensions

GitHub npm

Plug-and-play AI enablement framework for Node.js APIs. Transforms your endpoints into MCP tools and RAG-enabled knowledge sources using decorators. Works with Express, Fastify, Koa, Hono, NestJS, Next.js, tRPC, and AWS Lambda.

Installation

npm install @romatech/ai-extensions reflect-metadata

Quick Start

import 'reflect-metadata';
import express from 'express';
import {
    Controller, Get, Post, Delete,
    AiTool, AiHidden, AiDescription,
    useController, useAi
} from '@romatech/ai-extensions';

@Controller('/api/orders')
class OrdersController {
    @Get('/')
    @AiDescription('Lists all orders')
    static getAll(req, res) {
        res.json([{ id: 1, status: 'Pending' }]);
    }

    @Post('/')
    @AiTool('create_order')
    @AiDescription('Creates a new order')
    static create(req, res) {
        res.status(201).json({ id: 2, ...req.body });
    }

    @Delete('/:id')
    @AiHidden()
    static delete(req, res) {
        res.sendStatus(204);
    }
}

const app = express();
app.use(express.json());

useController(app, OrdersController);
useAi(app, { baseUrl: 'http://localhost:3000' });

app.listen(3000);

Features

Decorator-based

@AiTool, @AiHidden, @AiDescription, @AiCategory, @AiRateLimit — like .NET attributes.

Multi-Framework

Express, Fastify, Koa, Hono, NestJS, Next.js, tRPC, Lambda — auto-detected.

MCP Server

Full Model Context Protocol: tools, resources, prompts, completions at POST /mcp.

RAG Search

Built-in rag_search tool for semantic search across API documentation.

Auth & Rate Limiting

API key auth, CORS, per-tool and global rate limiting, role-based execution.

Circuit Breaker

Auto-disable failing tools. Multi-tenant support. Metrics and audit logging.

SSE + WebSocket

Persistent transport connections with heartbeat for real-time tool streaming.

Embedding Providers

Local (zero deps), OpenAI, or Ollama for RAG vector embeddings.

Swagger Discovery

Auto-discovers endpoints from OpenAPI spec. Custom handlers bypass HTTP.
